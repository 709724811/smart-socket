#Smart-Socket

##框架结构
- 传输层  
	smart-quickly:NIO的具体实现，并提供协议解析接口
- 协议层  
	1. smart-protocol-http:暂时未完善Http解析  
	2. smart-protocol-p2p:以二进制流进行点对点通信的协议，
- smart-omc 无效工程

##快速启动
服务端: P2PServer.java
客户端: P2PClient.java
